## Running React on Netlify

- [React](https://reactjs.org/) UI -is a popular JavaScript library for building user interfaces.
- [Vite](https://vitejs.dev/) FRONTEND - Hot Module Reloading (HMR), optimized builds, and TypeScript support out of the box.
- [Domains](https://bld.so/7OHY0) - this is where it all started
- [Multi-Site](https://www.geeksforgeeks.org/how-to-create-a-multi-page-website-using-react-js/) - React Router to create a multi-page website

### Getting Started w/ this repo

- Deploy contract & update "src/pages/labs/utils/contractABI.json" file
- Update CONTRACT_ADDRESS in "src/pages/labs/domainsite.jsx"
- Commit & Push, or Merge to dev & main branches to publish <https://nitsuah.io> automatically like <https://dev.nitsuah.io>
- Run `npm run dev` locally to get started
