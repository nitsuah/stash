# Auto — Car Project Board

A lightweight single-page HTML dashboard for tracking maintenance tasks and project status across multiple vehicles. Runs entirely in the browser with no backend — state is persisted via `localStorage`.

**Features:**
- Card-per-vehicle layout with status badge (Active / Soon / Sell)
- Checkbox task lists with strike-through on completion — persisted across page reloads
- Dark theme, responsive grid

**Usage:** Open `auto.html` directly in a browser. No build step or server required.
