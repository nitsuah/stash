# VMT - Vulnerability Management Tool

## Overview
Comprehensive IT asset and vulnerability management system built with VBA and Microsoft Access. Designed for IT security teams requiring centralized asset tracking, vulnerability remediation workflows, and compliance reporting.

## Key Features
- **Asset Management**: Complete IT equipment inventory and lifecycle tracking
- **Vulnerability Tracking**: End-to-end vulnerability assessment and remediation workflows
- **Change Management**: Structured change request processing with approval workflows
- **Progress Monitoring**: Real-time remediation status tracking and milestone reporting
- **Advanced Reporting**: Dynamic graphs, extended query tables, and API integration
- **Role-Based Access Control**: Granular user permissions and role management
- **Audit System**: Comprehensive event logging and compliance trail management

## Technical Architecture
- **Platform**: Microsoft Access + VBA
- **Core Modules**:
  - `import-comments.vb`: Bulk comment processing and workflow integration
  - `event-log.vb`: Real-time audit logging and compliance tracking

## Business Impact
- Centralizes vulnerability management across enterprise infrastructure
- Reduces remediation cycle time through automated tracking and notifications
- Ensures compliance with security frameworks through comprehensive audit trails
- Provides executive visibility into security posture and remediation progress

**Security Value**: Transforms manual vulnerability tracking into automated governance workflows, ensuring no security gaps fall through the cracks while maintaining detailed compliance documentation.
